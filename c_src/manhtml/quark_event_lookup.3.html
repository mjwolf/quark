NAME

quark_event_lookup — lookup a quark_event in quark's cache

SYNOPSIS

#include <quark.h>

int
quark_event_lookup(struct quark_queue *qq, struct quark_event *qev, int pid);

DESCRIPTION

quark_event_lookup looks for the cached event referenced by pid and fills qev with the result if successfull.

Quark's internal cache keeps processes that exited for a grace time, meaning you can still lookup them for a little while before they're garbage collected. At the time of this writing, this is hardcoded to 4 seconds.

RETURN VALUES

Zero if the event was cached and qev is filled. -1 otherwise and errno is set to ESRCH.
